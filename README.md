Der MINT-Kompass ist ein interaktiver Online-Test.
Du beantwortest 30 zufällig ausgewählte Fragen aus einem größeren Fragenpool.

Jede Frage hat 4 Antwortmöglichkeiten
Jede Antwort steht für einen MINT-Bereich:
🧮 Mathematik (M)
💻 Informatik (I)
🔬 Naturwissenschaft (N)
⚙️ Technik (T)

Am Ende bekommst du eine Auswertung in Prozent – sortiert von deinem stärksten bis schwächsten Bereich.
